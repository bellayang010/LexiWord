# LexiWord

An iOS vocabulary app for English language learners.
Scan any physical text with your camera → read it as clean e-book text →
tap any word to translate → save to your personal notebook.

## Features
- Camera OCR using Apple Vision framework
- Instant offline translation via Apple Translation framework
- Notion-style e-reader view
- Personal vocabulary notebook with dictionary definitions
- 100% on-device, no backend, no API costs

## Requirements
- iOS 17.4+
- Xcode 15+

## Setup
1. Clone the repo
2. Open LexiWord.xcodeproj
3. Set your development team in Signing & Capabilities
4. Run on device or simulator
